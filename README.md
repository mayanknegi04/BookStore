# 📚 Online E-Book Store

An Online E-Book Store is a web-based application that allows users to browse, search, and purchase digital books. The platform provides a seamless experience for readers to discover books, manage their accounts, and access purchased e-books online.

## 🚀 Features

### User Features
- User Registration and Login
- Browse Books by Category
- Search Books by Title, Author, or Genre
- View Book Details and Descriptions
- Add Books to Cart
- Purchase and Download E-Books
- User Profile Management
- Order History Tracking

### Admin Features
- Admin Dashboard
- Add, Update, and Delete Books
- Manage Categories
- Manage Users
- View Orders and Sales Information
- Inventory Management
